Лабораторна робота: CSS — виконання завдань

Завдання 1 — загальна сторінка

Цей документ демонструє різні способи використання CSS (inline, <style>, зовнішній файл), селектори, позиціонування, float, а також Flexbox-розташування елементів.

Завдання 2 — inline-стиль

Це приклад inline-стилю (атрибут style) — тут задано колір тексту, фон і відступи.

Завдання 3 — список з власним маркером

Нижче неупорядкований список використовує свій маркер (синій квадратик) через CSS.

  • Пункт з власним маркером — приклад 1
  • Пункт з власним маркером — приклад 2
  • Пункт з власним маркером — приклад 3
Завдання 4 — приклад float

Приклад тексту, який обтікає зображення праворуч або зліва, залежно від стилю. Тут використано властивість float у зовнішньому CSS. Після елемента, який плаває, бажано "очистити" потік, що показано наступним блоком з класом clearfix.

Завдання 5 — позиціонування
Це відносний контейнер (position: relative).
Абсолютно позиціонований (position: absolute)

Нижче — кнопка з фіксованим положенням (position: fixed) у правому нижньому куті.

Завдання 6 — контейнер з елементами (Flexbox)

Тут розташовано блоки, які наближено відповідають малюнку в умові (три колонки, центральний елемент — вертикально витягнутий).

Пояснення: для цього макета ми використали Flexbox із трьома колонками (дочірніми flex-елементами) і задали висоту центрального блока, щоб отримати вертикально витягнутий елемент згідно з ілюстрацією.

Додатково: використані селектори
  • Селектор по тегу: body, h1, p
  • Селектор по класу: .custom-marker, .flex-layout
  • Селектор по id: #main-container, #fixed-btn
