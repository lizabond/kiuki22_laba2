Лабораторна робота з CSS

2.3 Порядок виконання роботи

Створити HTML-документ із довільним вмістом, на якому під час лабораторної роботи демонструватиметься використання CSS. Має сенс одразу передбачити як блокові, так і рядкові елементи.

У документі продемонстровано такі способи підключення CSS:

  • Inline-стиль — застосований до елемента <body>.
  • Стильова секція <style> у <head>.
  • Зовнішній файл styles.css.

Використано селектори тегів, класів та ідентифікаторів. Додано елемент із використанням position. Додано елемент із використанням float. Додано елементи згідно варіанту — Flexbox фігура.
